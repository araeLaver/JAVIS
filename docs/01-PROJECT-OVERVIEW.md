# JAVIS 프로젝트 개요

## 프로젝트 비전

**목표**: 타사 API에 의존하지 않는, 완전히 내가 통제하는 개인 AI 비서 시스템

```
핵심 원칙:
├── 완전한 통제권: 모델, 데이터, 인프라 모두 내 소유
├── 지속적 발전: 버전업을 통한 꾸준한 모델 향상
├── 독립성: 외부 서비스 의존 최소화
└── 확장성: 작게 시작해서 점진적으로 확장
```

## 핵심 구성요소

### 1. JAVIS Model (나만의 모델)

| 항목 | 내용 |
|------|------|
| 베이스 모델 | Qwen2.5-7B-Instruct (Apache 2.0) |
| 파인튜닝 방식 | QLoRA (효율적, 저비용) |
| 서빙 | RunPod Serverless |
| 버전 관리 | 지속적 업데이트 (v0.1 → v0.2 → ...) |

### 2. JAVIS Core Engine (직접 개발)

- **대화 관리**: 멀티턴 대화, 컨텍스트 유지
- **도구 시스템**: 파일, 쉘, 웹, API 등 외부 도구 실행
- **메모리**: 단기(대화 내) + 장기(벡터 DB) 기억
- **RAG**: 개인 문서/지식 검색 및 활용
- **워크플로우**: 자동화 작업 실행

### 3. 학습 파이프라인 (지속적 발전)

```
사용 → 피드백 → 데이터화 → 재학습 → 배포 → 사용 ...
```

## 기대 효과

### 단기 (1-2개월)
- 나만의 말투와 스타일로 응답하는 AI
- 기본적인 작업 자동화 (파일, 쉘, 검색)

### 중기 (3-6개월)
- 복잡한 추론 및 멀티스텝 작업 수행
- 개인 지식베이스 활용 (RAG)
- 음성 인터페이스

### 장기 (6개월+)
- 완전한 개인 비서 시스템
- 자동화된 학습 파이프라인
- 확장 가능한 플러그인 생태계
